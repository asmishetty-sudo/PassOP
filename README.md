# Password Manager - React, Tailwind & MongoDB

A **secure and user-friendly Password Manager** built with **React**, **Tailwind CSS**, and **MongoDB**. This app allows users to **add, edit, and delete passwords**, store **URL, username, and password**, and manage their credentials efficiently.  

It also includes **React Toastify notifications**, **password visibility toggle**, and **animated icons** for a better user experience.
